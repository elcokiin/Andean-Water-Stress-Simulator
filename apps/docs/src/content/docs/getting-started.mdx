If you're new to this repository, follow these steps to get productive quickly.

<Steps>
  1. Install the workspace dependencies.

     <Tabs syncKey="pkg-manager">
       <TabItem label="bun">

       ```bash
       bun install
       ```

       </TabItem>
     </Tabs>

2.  Start the app you want to work on.

    <Tabs syncKey="workspace-app">
  <TabItem label="docs">

    ```bash
    bun run --filter @proyecto-final/docs dev
    ```

      </TabItem>
  <TabItem label="web">

    ```bash
    bun run --filter @proyecto-final/web dev
    ```

      </TabItem>
  <TabItem label="all">

    ```bash
    bun run dev
    ```

      </TabItem>
</Tabs>

3.  Use Turborepo scripts from the repository root.

         ```bash
         bun run build
         bun run lint
         bun run typecheck
         ```

    </Steps>
